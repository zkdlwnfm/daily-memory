현재 문제

Generic iOS 스타일링 — .systemBackground, .secondary 등 시스템 기본색만 사용. 브랜드 정체성 없음. "아무 앱이나 될 수 있는" 외형.

개선 방향

Ink Sage 브랜드 시스템 — dmPrimary(#3D5A50) 중심의 일관된 컬러 시스템. 앱을 열면 즉시 "Engram이다"라고 인지.

현재 문제

약한 비주얼 계층 — 폰트 weight에만 의존. 카드 그림자 opacity 0.03~0.04로 플랫하게 보임. 중요도 구분 불가.

개선 방향

3단계 엘레베이션 시스템 — Subtle(0.04)/Medium(0.08)/Emphasis(0.15) 그림자 + 배경 틴트로 깊이감.

현재 문제

빈 상태(Empty State)가 이모지+텍스트 — 📝 "No memories yet" 한 줄. 사용자에게 아무런 감정적 연결이나 행동 유도 없음.

개선 방향

일러스트 + 행동 유도 — 커스텀 일러스트레이션, 명확한 CTA 버튼, 격려 메시지로 첫 기록 유도.

현재 문제

스페이싱 비일관 — 4pt, 6pt, 8pt, 12pt, 16pt, 20pt, 24pt, 32pt, 48pt가 무작위 혼용. 시각적 리듬 없음.

개선 방향

8pt 그리드 시스템 — 8, 16, 24, 32, 40, 48만 사용. 일관된 시각적 리듬.

현재 문제

리텐션 훅 부재 — 스트릭은 있지만 시각화 약함. Year in Pixels 없음. 과거 기억 서피싱 알림 없음. 게이미피케이션 없음.

개선 방향

Daylio 수준 게이미피케이션 — Year in Pixels, 뱃지/업적, On This Day 플래시백, 무드 상관관계 분석.

핵심 UI 개선 항목

CRITICAL 홈 화면 리디자인

현재 홈은 평범한 카드 리스트. Structured 스타일 비주얼 타임라인으로 전환.

  • 상단: 인사말 + 오늘 날짜 + 스트릭 뱃지
  • 중단: 시간순 타임라인 (기억 + 리마인더 + 캘린더 통합)
  • 하단: 오늘의 무드 + 퀵 액션
  • Year in Pixels 미니 뷰 (월간 색상 그리드)
  • "On This Day" 플래시백 카드 (1년 전 기억)

CRITICAL 초저마찰 기록 UX

Daylio처럼 30초 내 기록 완성 가능한 퀵 엔트리 모드.

  • 무드 선택 (5단계 이모지 탭)
  • 활동 태그 (아이콘 멀티 셀렉트)
  • 한 줄 메모 (선택)
  • 사진 첨부 (선택)
  • "저장" → 완료. AI 분석은 백그라운드

CRITICAL 캘린더 히트맵

현재 캘린더는 숫자+점. Timepage/Fantastical 스타일 히트맵으로.

  • 기록 밀도를 색상 농도로 표현 (연한 sage → 진한 sage)
  • 무드 컬러 코딩 (행복=초록, 슬픔=파랑 등)
  • 일별 상세: 슬라이드업 시트 (바텀시트 패턴)
  • 월간/주간 토글

HIGH Year in Pixels

Daylio의 킬러 피처. 1년 전체를 색상 그리드로 시각화.

  • 12×31 그리드 (365일)
  • 각 셀 = 하루, 색상 = 무드/기록 여부
  • 빈 칸 = 회색 → "채우고 싶다" 충동 유발
  • Settings 또는 홈에서 접근

HIGH 스트릭 & 게이미피케이션

현재 스트릭은 텍스트 숫자만. 시각적 보상 시스템 필요.

  • 스트릭 링/서클 (Streaks 스타일)
  • 주간 7칸 채우기 진행바
  • 마일스톤 뱃지: 7일, 30일, 100일, 365일
  • 달성 시 축하 애니메이션 + 햅틱

HIGH On This Day 플래시백

Day One의 킬러 리텐션 피처. 과거 기억을 매일 서피싱.

  • 1년 전, 2년 전 같은 날 기억 카드
  • 푸시 알림: "1년 전 오늘, 민수랑 한강에서..."
  • 홈 화면 하단에 플래시백 섹션
  • 감정적 연결 → 알림 오픈율 극대화

MEDIUM 디자인 시스템 정비

하드코딩된 색상, 비일관적 스페이싱 → 토큰 기반 시스템.

  • ColorSystem: semantic naming (primary, surface, onSurface...)
  • SpacingScale: 8pt 그리드 (xs=4, sm=8, md=16, lg=24, xl=32)
  • TypographyScale: caption/body/title/headline
  • ShadowScale: subtle/medium/emphasis
  • CornerRadius: sm=8, md=16, lg=24, xl=32

MEDIUM 온보딩 리디자인

현재 시스템 아이콘+텍스트. Lottie 애니메이션 + 인터랙티브 온보딩.

  • 커스텀 일러스트레이션 (브랜드 아이덴티티)
  • 인터랙티브 데모: "직접 해보기" 단계
  • 마이크 권한 요청을 자연스럽게 통합
  • 첫 기록 유도 → 온보딩 완료 = 첫 기억 저장
