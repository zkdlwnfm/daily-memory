### 1.4 인터랙션

#### FAB 탭
```
1. Ripple 효과 (300ms)
2. Scale down 0.95 (100ms)
3. Scale up 1.0 (100ms)
4. 기록 화면으로 전환 (Slide up, 300ms)
```

#### 리마인더 카드 - 확인
```
1. 버튼 Ripple
2. 카드 Fade out + Slide left (300ms)
3. 리마인더 완료 처리
4. Snackbar: "리마인더를 완료했어요" [실행취소]
```

#### 리마인더 카드 - 미루기
```
1. 버튼 Ripple
2. Bottom Sheet 표시 (Slide up, 300ms)
   ┌─────────────────────────────────────┐
   │ 언제 다시 알려드릴까요?              │
   ├─────────────────────────────────────┤
   │ ○ 1시간 후                          │
   │ ○ 3시간 후                          │
   │ ○ 내일 오전 9시                     │
   │ ○ 직접 선택...                      │
   ├─────────────────────────────────────┤
   │         [취소]        [확인]         │
   └─────────────────────────────────────┘
```

#### 기록 카드 탭
```
1. Ripple 효과
2. 카드 elevation 증가 (2dp → 8dp, 100ms)
3. 기록 상세 화면으로 전환 (Shared element transition)
   - 카드 → 상세 화면으로 확장
```

#### Pull to Refresh
```
1. 화면 아래로 당김
2. 새로고침 인디케이터 표시
3. 데이터 로드 (최대 3초)
4. 인디케이터 사라짐
5. 새 데이터 Fade in
```

### 1.5 애니메이션

#### 화면 진입
```
순서대로 Fade in + Slide up:
1. Header (0ms)
2. Greeting (100ms)
3. Reminder Card (200ms)
4. Recent Memories (300ms)
5. Flashback (400ms)
6. FAB (500ms)

각 요소: duration 300ms, ease-out
```

#### 실시간 업데이트
```
새 기록이 추가되면:
1. 기존 카드들 아래로 밀림 (Slide down, 300ms)
2. 새 카드 Fade in + Slide down from top (300ms)
3. Greeting 숫자 업데이트 (Count up animation)
```
