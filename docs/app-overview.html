Core Features

Everything the app can do

🎤
Voice & Text Recording
Capture memories naturally through speech or typing
  • Real-time speech recognition (SFSpeechRecognizer)
  • Live transcription with audio level visualization
  • Text mode with direct input
  • Attach up to 5 photos per memory
  • GPS location tagging on save
🤖
AI Analysis
GPT-4o extracts structured data from your memories
  • Person extraction with relationship inference (Family, Friend, Colleague...)
  • Location detection
  • Date and event recognition
  • Financial amount extraction
  • Auto-categorization (Event, Promise, Meeting, Financial, General)
  • Image analysis via GPT-4o Vision (objects, scene, OCR, faces)
🔎
Semantic Search
Ask natural questions about your memories
  • Vector embeddings (OpenAI text-embedding-3-small, 1536 dims)
  • Cosine similarity via pgvector on PostgreSQL
  • Hybrid search: semantic + keyword fallback
  • Natural language answers ("When did I last see John?")
  • Auto-generated follow-up questions
  • Suggestion prompts for new users
👥
People Tracking
Automatically build and maintain your relationship network
  • Auto-extracted from AI analysis with relationship type
  • Meeting count & last contact date tracking
  • Contact warning if 30+ days without interaction
  • Profile photo, contact info, personal memo
  • Sort: Recent, A-Z, Most frequent
  • Per-person memory timeline
🔔
Smart Reminders
Never forget promises, events, or important dates
  • Time-based: one-time, daily, weekly, monthly, yearly
  • Location-based: geofencing with enter/exit/both triggers
  • Adjustable radius (50-500m)
  • Quick set: In 1 hour, Tomorrow, Next week
  • Complete & Snooze (15min) actions from notification
  • Link to memory or person
☁️
Cloud Sync
Offline-first with automatic cloud synchronization
  • Firebase Auth (Email, Apple, Google sign-in)
  • Firestore: memories, persons, reminders sync
  • Firebase Storage: photo backup
  • Offline queue with auto-sync on reconnect
  • Conflict resolution (latest update wins)
  • Network monitoring (NWPathMonitor)

System Architecture

How everything connects

iOS App (SwiftUI) | |-- Firebase Auth -----> Authentication (Email, Apple, Google) |-- Firebase Firestore -> Cloud sync (memories, persons, reminders) |-- Firebase Storage --> Photo backup |-- Firebase Analytics -> Usage tracking, Crashlytics | |-- NestJS API Server --> https://dailymemory.pjhdev.co.kr | | | |-- POST /ai/analyze ---------> OpenAI GPT-4o-mini (text analysis) | |-- POST /ai/analyze-image ---> OpenAI GPT-4o Vision (image analysis) | |-- POST /embeddings/store ---> OpenAI text-embedding-3-small | |-- POST /search/semantic ----> pgvector cosine similarity | | | |-- PostgreSQL 16 (shared-db) | | |-- memory_embeddings (vector 1536) | | |-- user_profiles | | |-- api_usage_logs | | | |-- Redis 6 (shared-db) | |-- Rate limiting (per-user daily counters) | |-- Embedding cache | |-- Local Storage |-- CoreData (offline-first) |-- VectorSearchEngine (local fallback) |-- Photo files + thumbnails

Backend API

NestJS server on Proxmox LXC 304 - all endpoints require Firebase Bearer token

Method Endpoint Description AI Model
GET /api/v1/health Health check (no auth required) -
POST /api/v1/ai/analyze Extract persons, location, date, amount, category from text gpt-4o-mini
POST /api/v1/ai/analyze-image Analyze image: objects, scene, OCR, faces, tags gpt-4o
POST /api/v1/embeddings Generate vector embedding for text text-embedding-3-small
POST /api/v1/embeddings/store Generate + store embedding for a memory (pgvector) text-embedding-3-small
POST /api/v1/embeddings/batch Batch embedding generation (up to 20 texts) text-embedding-3-small
POST /api/v1/search/semantic Semantic search with cosine similarity threshold text-embedding-3-small + pgvector
DELETE /api/v1/embeddings/:memoryId Remove stored embedding -

Data Models

Core domain entities

📋
Memory
  • content (text), photos[] (up to 5)
  • extractedPersons[], extractedLocation, extractedDate
  • extractedAmount, extractedTags[]
  • category (EVENT/PROMISE/MEETING/FINANCIAL/GENERAL)
  • importance (1-5), isLocked, excludeFromAI
  • embedding (vector 1536), GPS coordinates
  • syncStatus (SYNCED/PENDING/CONFLICT/LOCAL_ONLY)
👤
Person
  • name, nickname, profileImageUrl
  • relationship (FAMILY/FRIEND/COLLEAGUE/BUSINESS/ACQUAINTANCE/OTHER)
  • phone, email (optional contact info)
  • summary (AI-generated)
  • meetingCount, lastMeetingDate (auto-calculated)
  • memo (personal notes)
🔔
Reminder
  • title, body, scheduledAt
  • repeatType (NONE/DAILY/WEEKLY/MONTHLY/YEARLY)
  • Location: latitude, longitude, radius (50-500m)
  • locationTriggerType (ENTER/EXIT/BOTH)
  • Links: memoryId, personId (optional)
  • isActive, isAutoGenerated, triggeredAt
