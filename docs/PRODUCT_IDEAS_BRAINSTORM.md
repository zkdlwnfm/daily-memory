## 🏆 Top 5 우선순위 아이디어

전략적 정렬, 임팩트, 실행 가능성, 차별화를 종합 평가했습니다.

### 1. 🥇 온보딩 "첫 기록 3개" 챌린지 (PM3)

**Why**: 리텐션의 가장 큰 적은 "첫 주에 습관이 안 잡히는 것". 가장 적은 노력으로 가장 큰 리텐션 개선 기대.

- **구현**: 온보딩 플로우 + 진행 카드 (0/3) + 완료 축하 애니메이션
- **Effort**: 2-3일
- **검증할 가정**: "구조화된 온보딩이 7일 리텐션을 20%p 이상 올린다"
- **측정**: Day 1 기록률, Day 7 복귀율

### 2. 🥈 주간 메모리 리포트 + "1년 전 오늘" 강화 (PM1 + PM2)

**Why**: 사용자가 앱을 열지 않아도 가치를 전달하는 유일한 채널 = 푸시 알림. 감성적 리마인더는 DAU를 끌어올리는 가장 검증된 패턴.

- **구현**: Airflow DAG (이미 skeleton 존재) + 푸시 알림 템플릿 + 홈 플래시백 UI 개선
- **Effort**: 3-5일
- **검증할 가정**: "주간 리포트를 받은 사용자의 주간 방문이 2배 이상 높다"
- **측정**: 알림 오픈율, 알림 → 앱 세션 전환율

### 3. 🥉 원탭 음성 기록 + 잠금화면 위젯 (D1)

**Why**: 핵심 가치가 "기록 마찰 최소화"인데, 현재 앱 열기 → 기록 탭 → 녹음 버튼 3단계. 1단계로 줄이면 핵심 가치 직접 강화.

- **구현**: Lock Screen Widget (iOS 16+) + App Intent + 바로 녹음 시작
- **Effort**: 3-4일
- **검증할 가정**: "위젯 사용자의 일일 기록 수가 2배 이상 높다"
- **측정**: 위젯 탭 수, 위젯 사용자 vs 비사용자 기록 빈도

### 4. 감정 태깅 + 무드 트래커 (D2)

**Why**: 일기앱 사용자의 #1 니즈는 "내 감정을 돌아보기". 이모지 하나로 구현이 간단하면서, 감정 통계라는 새로운 가치를 창출.

- **구현**: Memory 모델에 mood 필드 추가 + 기록 후 이모지 선택 UI + 월별 감정 차트
- **Effort**: 2-3일
- **검증할 가정**: "감정 태깅 사용자의 기록 빈도가 30% 이상 높다"
- **측정**: 태깅 사용률, 감정 통계 페이지 조회수

### 5. AI 대화형 검색 (E3)

**Why**: 현재 시맨틱 검색이 1회성 질문/응답. 대화형으로 바꾸면 "AI가 기억해준다"는 핵심 가치를 극대화. 경쟁 앱 대비 최대 차별화.

- **구현**: SearchView에 채팅 UI + 이전 질문 컨텍스트를 GPT에 전달 + 관련 메모리 인라인 표시
- **Effort**: 5-7일
- **검증할 가정**: "대화형 검색 사용자의 월간 검색 횟수가 3배 이상"
- **측정**: 검색 세션당 질문 수, 검색 후 메모리 열람률
