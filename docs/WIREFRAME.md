# DailyMemory - UI/UX Wireframe

## 디자인 원칙

1. **기록 마찰 최소화** - 3초 안에 기록 시작 가능
2. **AI는 보조자** - AI 결과는 항상 수정 가능, 사용자가 최종 결정
3. **정보 계층** - 중요한 것부터 보여주기
4. **일관된 패턴** - 비슷한 동작은 비슷한 UI

### 인터랙션
- FAB 탭 → 기록 화면 (음성 모드로 시작)
- 기록 카드 탭 → 기록 상세
- 리마인더 [확인] → 완료 처리
- 리마인더 [미루기] → 시간 선택 팝업

## 9. 주요 User Flow

### 9.1 빠른 음성 기록 Flow

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│  위젯    │ →  │  녹음    │ →  │ AI 분석  │ →  │   저장   │
│  🎤 탭   │    │  중...   │    │  결과    │    │   완료   │
└──────────┘    └──────────┘    └──────────┘    └──────────┘
                    │                │
                    │   말 멈추면    │   확인만 하고
                    │   자동 종료    │   바로 저장
                    ▼                ▼
               [3초 이내]       [2초 이내]

총 소요 시간: 녹음 시간 + 약 5초
```

### 9.2 AI 검색 Flow

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│  검색탭  │ →  │  질문    │ →  │ AI 응답  │ →  │  기록    │
│          │    │  입력    │    │  표시    │    │  상세    │
└──────────┘    └──────────┘    └──────────┘    └──────────┘
                                     │
                              관련 기록 카드
                              탭하면 상세로
```

### 9.3 리마인더 Flow

```
┌──────────┐    ┌──────────┐    ┌──────────┐
│  푸시    │ →  │   홈     │ →  │   기록   │
│  알림    │    │ 리마인더 │    │   상세   │
└──────────┘    └──────────┘    └──────────┘
     │              │
     │         [확인] → 완료
     │         [미루기] → 시간 선택
     ▼
  탭하면 앱 열림
```
